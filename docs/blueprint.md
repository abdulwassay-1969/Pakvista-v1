# **App Name**: Explore Pakistan

## Core Features:

- Hero Section: Display a full-screen scenic image with a dark gradient overlay, heading, subtitle, curved SVG wave, and a transparent navbar.
- Districts Slider: Create a responsive image slider/grid hybrid to showcase districts of Pakistan.
- District Card Design: Design a reusable DistrictCard component with a background image, dark overlay, district name, hover zoom effect, and click-through functionality.
- Province Navigation: Navigation of the provinces and linking the displayed information about the regions to detail pages.
- Tourism Intro Section: Add an introductory text section about Tourism in Pakistan to familiarize users with Pakistani tourism.
- Events Section: Display a list of destinations highlights in a specific event.

## Style Guidelines:

- Primary color: Dark Blue (#1E3A8A) for an official, premium feel.
- Background color: Very light gray (#F9FAFB) for a clean, spacious layout.
- Accent color: Light blue (#38BDF8) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look.
- Use Lucide icons for a consistent and modern appearance.
- Implement a clean, spacious layout with a focus on readability and visual hierarchy.
- Incorporate smooth fade-in on scroll and image zoom on hover to enhance UX.
